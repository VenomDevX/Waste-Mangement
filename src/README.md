# WasteWise India - Smart Waste Management System

A comprehensive waste management web application built for the Smart India Hackathon (SIH) addressing Problem Statement ID 25060 on "Real life solutions for Waste Management".

## 🎯 Project Overview

This React-based application implements key solutions for waste management including:

- **Mandatory Citizen Training** - Interactive modules for waste management education
- **Green Champions Program** - Community monitoring and reward system  
- **Digital Facility Tracking** - Waste facility locator and vehicle tracking
- **Community Participation** - Social features and engagement tools
- **Reporting System** - "If you see waste, send photo" reporting initiative
- **National Dashboard** - Real-time waste management statistics

## 🏗️ Architecture

Built with modern React and TypeScript, using:
- **Tailwind CSS V4** - For responsive, utility-first styling
- **ShadCN/UI Components** - For consistent UI elements
- **Component-based Architecture** - Modular and maintainable codebase

## 📱 Features

### Core Sections
1. **Hero Section** - Landing with key statistics
2. **Dashboard** - National waste management metrics  
3. **Training** - Mandatory education modules
4. **Facilities** - Location finder and utility shopping
5. **Community** - Social engagement and leaderboards
6. **Reporting** - Waste issue reporting system

### Government Integration
- Designed for government deployment
- Green color scheme following environmental themes
- Compliance with digital governance standards
- Accessibility and responsive design

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Open in browser at `http://localhost:3000`

## 🎨 Design System

Uses Tailwind V4 with custom CSS variables defined in `/styles/globals.css`:
- Consistent color palette
- Typography system
- Component tokens
- Dark mode support
