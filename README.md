# WasteWise India

WasteWise India is a modern, responsive web application designed to track, manage, and promote scientific waste treatment across the nation. The platform provides a centralized dashboard for national statistics, interactive educational modules, and real-time insights into state-wise progress.

## 🚀 Features

- **National Dashboard:** Real-time metrics on waste treatment, tonnes/day handled, and Waste-to-Energy (WTE) plant distribution.
- **Educational Quizzes:** Quick interactive modules to test and improve citizen knowledge on proper waste segregation.
- **Responsive Design:** A premium, fully responsive interface built with React and Tailwind CSS that adapts seamlessly to desktop, tablet, and mobile devices.
- **Action-Oriented CTAs:** Tools to schedule waste pickups and locate local waste management facilities.

## 🛠️ Technology Stack

- **Frontend Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (Custom Responsive Grids & Layouts)
- **Icons:** Lucide React
- **Language:** TypeScript

## 🏃‍♂️ Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/VenomDevX/Waste-Mangement.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd "Waste Management Solutions Website"
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **View the app:**
   Open your browser and navigate to `http://localhost:3000` (or the port specified by Vite in your terminal).
